##动作栏

动作栏，运行在你的activity之上，是你的UI的骨干。在这里，你可以给用户提供与当前activity相关的动让它执行(例如，修改你查看的联系人)或者与整体的应用相关(例如，"这里是文档")。有时候，这些动作会以工具栏按钮或其他控件的形式出现在动作栏中。有时候，这些动作会出现在“overflow”，这就相当于一个菜单。

这章介绍了动作栏的概念以及如何在其之上添加动作。
